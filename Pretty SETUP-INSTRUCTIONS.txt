SETUP INSTRUCTIONS — TeacherDashboard
======================================
Come back to this when you're ready to install Node.js and run the code formatting tools.

STEP 1: Install Node.js
------------------------
1. Open your browser and go to: https://nodejs.org
2. Click the big green LTS button on the left side
3. Open the downloaded .pkg file
4. Click through the installer (just keep clicking Continue/Install)
5. QUIT Terminal completely (Cmd+Q) and reopen it

Verify it worked:
  node --version    (should show something like v22.x.x)
  npm --version     (should show something like 10.x.x)

STEP 2: Install project dependencies
-------------------------------------
  cd ~/TeacherDashboard
  npm install

This reads package.json and installs Prettier (the code formatter) and Vitest (the test runner).

STEP 3: Format the codebase
----------------------------
  npm run format

This runs Prettier across all JS and CSS files so the code style is consistent.

STEP 4: Run tests
------------------
  npm test

This runs the gb-calc.js test suite. All 47 tests should pass.

STEP 5: Run locally (optional)
-------------------------------
  npx serve . -p 8090

Opens a local server at http://localhost:8090
You'll be redirected to login since Supabase auth is required.

WHAT THESE TOOLS DO
====================
- Node.js: JavaScript runtime that lets you run tools outside the browser
- npm: Package manager that comes with Node.js (like an app store for dev tools)
- Prettier: Auto-formats code so it looks consistent (spaces, quotes, line breaks)
- Vitest: Runs automated tests to verify the calc engine works correctly
- npx serve: Simple local web server for testing
